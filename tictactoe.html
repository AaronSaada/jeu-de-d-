Jeu du Morpion

Règles du jeu

Nombre de joueurs :

Le jeu se joue à 2 joueurs.

Objectifs :

Le 1er joueur à aligner 3 de ses signes gagne la partie.

Déroulement du jeu :

  • Le joueur 1 contrôle les 'X', le joueur 2 contrôle les 'O'.
  • Tour à tour, les joueurs place leur signe dans une case vide.
  • Le 1er joueur à aligner 3 de ses signes gagne la partie.
